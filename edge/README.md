# PiLambdaChart Edge Client

This folder contains the Python agent running on the Raspberry Pi edge device.

## Folder Structure
*   `src/agent.py`: Asynchronous main event loop running as a systemd daemon.
*   `src/sensors/`: Directory for hardware sensor interfaces.
*   `requirements.txt`: Python package dependencies.
*   `sensor_check`: Python scripts that check individual sensors for their connection and basic functionality.
